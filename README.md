# agile-user-stories-ecommerce-qa
User Story

Descrição:
Projeto de criação de User Stories com foco na definição de requisitos funcionais para um sistema de e-commerce.

Foram elaboradas histórias de usuário com critérios de aceitação claros, garantindo alinhamento entre negócio e desenvolvimento.

Este projeto demonstra entendimento de requisitos e colaboração com metodologias ágeis.
